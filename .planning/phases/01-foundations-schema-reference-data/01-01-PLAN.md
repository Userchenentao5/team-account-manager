This plan creates the load-bearing schema symbols the rest of the project builds against:

**Drizzle tables (`src/db/schema.ts`):**
- `currency` ŌĆö columns: `code` (text PK; 'USD','CNY','EUR','GBP','JPY','HKD'), `name` (text), `minor_unit` (integer; ISO-4217 exponent ŌĆö JPY=0, others=2), `is_active` (boolean).

- `space` (DECLARED NOW, populated Phase 3) ŌĆö columns: `id` (PK), `name` (text), `country` (text; ISO-3166 alpha-2, D-11), `payment_channel_id` (integer FK ŌåÆ payment_channel.id, D-07), `currency_code` (text FK ŌåÆ currency.code), **`amount_minor`** (integer minor units), **`period_unit`** (text 'month'|'quarter'|'year'), **`period_count`** (integer) ŌĆö structured period `{unit,count}`; FX-snapshot LOCKED-NAME columns **`rate_used`** (text decimal string), **`rate_as_of`** (text), **`rate_source`** (text), **`amount_usd`** (integer USD minor units); `opening_date` (text YYYY-MM-DD), `expiry_date` (text).

**Constants & helpers:**
- `src/lib/countries.ts` ŌåÆ exported `COUNTRIES` ISO-3166 alpha-2 constant (code + label list).
- `src/lib/money.ts` ŌåÆ `formatMinor(amountMinor, exponent)` and `parseToMinor(input, exponent)`.

**Currency seed set (D-02/D-03):** USD(2), CNY(2), EUR(2), GBP(2), JPY(0), HKD(2).

**Runner artifacts:** `src/db/index.ts` (db singleton), `src/db/migrate.ts`, `src/db/seed.ts`, committed `drizzle/*.sql`.
