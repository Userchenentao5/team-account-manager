实现反腐层 FX 服务:唯一与 Frankfurter 通信的模块。它以 `AbortSignal.timeout` 拉取 `base=USD` 汇率,用 Zod 校验外部响应(ASVS V5 信任边界),将 USD→X 反转为 X→USD 十进制字符串(USD 固定为 "1"),原子 upsert,并在任何失败时降级到上次缓存并置 `stale:true` —— 绝不写入 0/NULL。

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Frankfurter API → FX service | The external HTTP response is untrusted input; it must be validated before touching the cache. |
| outbound HTTP (app → api.frankfurter.dev) | Fixed HTTPS URL, time-bounded request; no user-controlled URL (no SSRF surface). |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-02-01 | Tampering | external response → upsertRates | high | mitigate | Zod validate-then-upsert atomically (ASVS V5); reject 0/negative/NaN/Infinity; on any throw write nothing and fall back to cache (Pitfall 1). |
| T-02-03 | Tampering | invertToUsd | high | mitigate | Guard rejects non-finite/non-positive input by throwing; prevents division-by-zero / Infinity poisoning Phase 3 USD math. |
| T-02-02 | Denial of Service | refreshFromApi / ensureFreshRates | medium | mitigate | `AbortSignal.timeout(4000)` bounds the call; failure degrades to cache + stale flag, never throws to the caller (FX-03). |
| T-02-09 | Information Disclosure | outbound fetch | low | accept | Fixed HTTPS URL to api.frankfurter.dev; keyless API, no secrets/PII transmitted (ASVS V9). |
</threat_model>
