## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-02-01 | Tampering | upsertRates / fx_rate | high | mitigate | All rows written in one `db.transaction`; on any throw the cache is left untouched — never a partial/0/NULL row set (Pitfall 1). |
| T-02-04 | Tampering | fxRates.ts queries | low | mitigate | Drizzle parameterized builders only; no string-concatenated SQL — same discipline as channels.ts. |
| T-02-IN | Tampering | fx_rate FK | low | mitigate | `currencyCode` FK → `currency.code` constrains cache rows to the 6 seeded currencies; FK enforcement ON. |
