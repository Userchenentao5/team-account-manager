## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser confirmation to Server Action | Disabled UI can be bypassed, so confirmation text must be validated server-side. |
| Server Action to database transaction | Destructive mutation crosses into persistent storage. |

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |

| T-04-04-TAMPER | Tampering | `deleteSpace` action | high | mitigate | Parse `{ id, confirmationName }`, fetch current row, and reject exact-name mismatch server-side. |
