## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser to Server Action | User-controlled form payloads cross into trusted mutation code. |
| Server Action to FX cache | Monthly USD snapshots depend on cached rate rows and currency exponents. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-04-02-TAMPER | Tampering | `createChildAccount` and `updateChildAccount` | high | mitigate | Re-parse with Zod, validate owning space/child IDs, and whitelist persisted fields. |
| T-04-02-INFO | Information Disclosure | child account action payloads | high | mitigate | Schema excludes credential-like fields and tests assert they are not persisted. |
| T-04-02-REPLAY | Repudiation | child snapshot edits | medium | mitigate | Preserve snapshot on non-price edits and refreeze only on amount/currency changes with tests. |
| T-04-02-DOS | Denial of Service | missing FX rate path | medium | mitigate | Return a clear no-rate error before writing when `getRate` is missing. |
| T-04-02-SC | Tampering | npm installs | low | accept | No package install or upgrade is required. |
</threat_model>
