## Critical Anti-Patterns (do NOT repeat these)

- Do not aggregate `space.amountUsd` through a child-account join. Aggregate parent space payments and child monthly costs separately, then merge integer subtotals.
- Do not recompute dashboard USD totals from live FX cache rows, `rateUsed`, `amountMinor`, or conversion helpers. Frozen stored USD values are authoritative.
- Do not add dashboard mutation controls. Phase 5 links to existing detail/list flows only.
- Do not add a chart package or third-party dashboard block. Use CSS bars and existing UI primitives.
- Do not use Jest-only flags such as `--runInBand`; this project uses Vitest.
