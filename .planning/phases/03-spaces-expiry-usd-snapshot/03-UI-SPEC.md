## Spacing Scale

8-point scale — matches the Tailwind utilities already used across the channel/rate slices.

| Token | Value | Tailwind | Usage (established) |
|-------|-------|----------|---------------------|
| xs | 4px | `gap-1` | Icon-button cluster gaps in a table action cell |
| sm | 8px | `gap-2` | Switch + label, inline label/control pairs |
| md | 16px | `gap-4` | Header row gaps, empty-state inner stack |
| lg | 24px | `gap-6` / `p-6` | **Default**: page wrapper padding, top-level section stack, header-control group |
| xl | 32px | `gap-8` | Reserve for form column gaps on the create/edit page if needed |
| 2xl | 48px | `py-12` | Empty-state vertical padding inside dashed card |
| 3xl | 64px | `p-16` | Page-level breathing room (rarely needed; tool is dense) |

Page shell pattern (mandatory, mirrors `rate-table.tsx` / `channel-table.tsx`):
`<div className="flex flex-col gap-6 p-6">` with a `flex items-center justify-between gap-4` header row.

Exceptions:
- Icon-only action buttons use `size-11` (44px) for touch-target accessibility while the icon glyph stays `size-4` (16px). This is the established channel-table pattern and is the one allowed non-scale dimension.

## Typography

Extracted from shipped components. Exactly 3 sizes in active use plus an inherited 16px base; exactly 2 weights.

| Role | Size | Weight | Line Height | Tailwind (established) |
|------|------|--------|-------------|------------------------|
| Page heading (h1) | 24px | 600 (semibold) | tight (~1.2) | `text-2xl font-semibold leading-tight` |
| Empty-state / section heading (h2) | 18px | 600 (semibold) | default (~1.4) | `text-lg font-semibold` |
| Body / labels / captions | 14px | 400 (regular) | default (1.5) | `text-sm` (+ `text-muted-foreground` for secondary) |
| Base / form inputs | 16px | 400 (regular) | 1.5 | inherited (`html { font-sans }`) — input control text |

Weights: regular (400) + semibold (600) only. No other weights.

Numeric / code rule: currency codes, amounts (original + frozen USD), rates, and dates render in `font-mono` right-aligned in tables (`className="text-right font-mono"`) — mirrors `rate-table.tsx`. Money major-unit display uses existing `formatMinor`; alignment via `text-right`.

## Color

The shipped palette is a **strict neutral OKLCH grayscale** (base color `neutral`); chart tokens are grayscale too. There is no decorative brand hue. Color is spent almost entirely on neutrals, with `destructive` (red) as the only saturated semantic. The expiry tri-state (D-07) introduces one new semantic warning hue — defined below — because expiry status is the single place where signal color carries Core Value.

| Role | Value (token) | Usage |
|------|---------------|-------|
| Dominant (60%) | `--background` `oklch(1 0 0)` light / `oklch(0.145 0 0)` dark, `--foreground` text | Page background, table surface, primary text |
| Secondary (30%) | `--muted` / `--muted-foreground` / `--secondary` / `--sidebar` | Sidebar, muted captions/labels, secondary buttons, archived row tint (`bg-muted/50`), dashed empty-state border |
| Accent (10%) | `--primary` `oklch(0.205 0 0)` (near-black) | See reserved list below |
| Destructive | `--destructive` `oklch(0.577 0.245 27.325)` | Save/refresh failure alerts + toasts, and the **expired** expiry badge |
| Warning (new semantic) | amber — `oklch(0.769 0.15 70)` light / `oklch(0.79 0.14 70)` dark | **Only** the "即将到期 (≤7 天)" expiry badge. Add as a scoped class on `expiry-badge.tsx`; do not promote to a global accent. |

Accent reserved for: the **primary CTA button fill** (`新增空间`, form `保存`) and the active sidebar nav item. Nothing else. (shadcn `--accent` token itself is a subtle neutral hover surface, `menuAccent: subtle` — it is hover state, not a signal color.)

Expiry tri-state badge color mapping (D-07, display-only, never stored):
- `expired` (days < 0) → `Badge variant="destructive"` (red) — label `已过期`
- `soon` (0 ≤ days ≤ 7) → amber warning badge (scoped class above) — label `即将到期`
- `normal` (days > 7) → `Badge variant="outline"` (neutral) — label `正常`

Status must never rely on color alone — each badge carries its text label (accessibility).

## Copywriting Contract

Language: Chinese (zh-CN) with English technical terms retained (USD, codex, chatgpt), matching shipped copy and the user's stated language preference. Tone: concise, direct, no exclamation.

| Element | Copy |
|---------|------|
| Primary CTA (list) | `新增空间` (with `Plus` icon, `size-4`) |
| Primary CTA (form submit) | `保存` — pending state: `保存中…` |
| Secondary action (form) | `取消` |
| Edit affordance (row) | icon button, `aria-label="编辑 {空间名称}"`, `Pencil` icon, tooltip `编辑` |
| Empty state heading | `还没有空间` |
| Empty state body | `创建你的第一个空间,记录归属国家、支付渠道、金额与订阅周期,系统会自动算出到期日并冻结 USD 金额。` (+ inline `新增空间` CTA) |
| Filtered-empty (filters active, 0 results) | heading `没有符合条件的空间` / body `调整或清除筛选条件后再试。` (no create CTA — distinct from true-empty) |
| Error state — blocked save (D-02, no cached rate) | `该币种暂无汇率,无法折算 USD。请先到「汇率」页刷新汇率后重试。` (block, toast.error + inline form error) |
| Error state — generic action failure | `保存失败,请重试。` (mirrors channel/rate slices) |
| Stale-rate notice on save (optional, D-09 / RESEARCH OpenQ2) | non-blocking `toast`: `已使用最近一次缓存的汇率折算。` — save still succeeds |
| Success — create | `已创建空间` |
| Success — edit | `已保存修改` |
| As-of / frozen-rate label (detail) | `汇率截至 {yyyy-MM-dd HH:mm}（已冻结）` — mirrors `汇率截至` from rate-table |
| Child-accounts placeholder (detail, Phase 4 deferred) | heading `子账号` / body `子账号管理将在后续版本提供。` |

Destructive actions this phase: **none.** Space deletion + cascade is Phase 4 (SPACE-05). No delete/archive control ships in this phase. (If an `alert-dialog` confirm is later needed, follow the channel `archive-dialog.tsx` pattern: reversible → `variant="secondary"` confirm; irreversible → `variant="destructive"`.)

## Interaction Contract

| Interaction | Pattern (established) |
|-------------|------------------------|
| List sort/filter | URL search params drive RSC re-query (`router.push("/spaces?country=..&channel=..")`), exactly like channel-table `?archived=1`. Default sort: expiry ascending (D-06). |
| Create / edit | `react-hook-form` + `zodResolver` (client UX) → Server Action re-parse (trust boundary). Submit via `useTransition`; result `{ok, error}` → `setError` / `toast`. Create may be a `dialog` or a `/spaces/new` page — Claude's discretion (CONTEXT). |
| Pending state | Disable submit/refresh button during its own transition; spinner = `RefreshCw`/icon with `animate-spin`, label swaps to `…中…`. |
| Toast | `sonner` `toast.success` / `toast.error` after every action (success + every failure branch). |
| Empty vs filtered-empty | Two distinct states — true-empty shows create CTA; filtered-empty shows "clear filters" guidance, no CTA (mirrors rate-table's stale-vs-empty distinction). |
| Detail page | RSC read; renders mother account (email/login), expiry date + tri-state badge, original amount (`font-mono`), frozen USD amount (`font-mono`), as-of/frozen label, and the deferred child-accounts placeholder block. |
